## 기능
1. mongodb와 연결
2. aws ec2에 배포
3. 상품(이름, 날짜, 판매 상태, 작성자, 비밀번호) 생성
4. 상품 목록(전체, id를 통한 상세 내용) 조회
5. 상품 정보(내용, 판매 상태) 수정
6. 상품 삭제
